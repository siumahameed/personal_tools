## API Endpoints

All under `/api/hub/`:

| Method | Path | Return |
|--------|------|--------|
| GET | /api/hub/list | Array of 8 scholarship summaries (id, name, country, deadline, progress_pct) |
| GET | /api/hub/{id} | Full scholarship knowledge document |
| GET | /api/hub/{id}/roadmap | Personalized roadmap (generated by GuidanceEngine) |
| POST | /api/hub/{id}/readiness | Update readiness progress |
| POST | /api/hub/{id}/applicants | Add a known past applicant manually |
| GET | /api/hub/{id}/applicants | List scraped + manual past applicants |

## UI Design

### Hub Overview Page (`/hub`)
- Simple scrollable grid of 8 cards
- Each card shows: name, country, funding type, deadline with countdown, readiness progress bar
- Color-coded: green (ready), yellow (in progress), red (not started)
- Click any card → navigates to `/hub/{id}`

### Hub Detail Page (`/hub/{id}`)
Single scrollable page with collapsible sections:

1. **Header** — Scholarship name, tags (country, funding), back button
2. **Your Readiness** — Progress bar with percentage, update button
3. **Eligibility Check** — Per-criteria flags (✅ pass / ❌ fail / ⚠️ needs attention), computed against user profile from config.py
4. **Step-by-Step Roadmap** — 5 phases, each expandable to show individual steps with checkboxes
5. **Past Successful Applicants** — Cards showing name, background, tips. Add button for manual entry.
6. **Tips & Tricks** — Bulleted list per scholarship. Add note button.
7. **Documents Needed** — Checklist with status indicators

## GuidanceEngine

Generates personalized roadmap by considering:
- User's GPA vs scholarship minimum
- User's graduation timeline vs deadline
- User's country (Bangladesh-specific requirements)
- Missing documents (language test, LORs, etc.)
- Competition level (EMJM 5%, Chevening 0.7% acceptance)

Output: 5-phase roadmap with estimated timelines, personalized tips, and priority actions.

## ApplicantScraper

- Takes a scholarship ID
- Searches preset URLs (scholarship alumni pages, blogs, forums)
- Uses BeautifulSoup to extract applicant profiles
- Falls back gracefully if no data found
- Results stored alongside manual entries in the knowledge document
