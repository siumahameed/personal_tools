General Documents Required
  • Transcripts & Degree — Certified copies. English translation if needed.
  • CV / Resume — Europass format for Erasmus. 1-2 pages.
  • Motivation Letter / SOP — 1-2 pages: background, goals, why this programme.
  • Reference Letters — 2-3 from professors or employers. Signed, letterhead, dated.
  • English Proficiency — IELTS 6.5+, TOEFL 90+, or PTE.
  • Passport Copy — Valid 2+ years.
  • Portfolio / Video — Required by some programmes.

MBZUAI — Mohamed Bin Zayed University of AI
Click card to see full details
Documents Required
  • Online application via MBZUAI portal
  • Bachelor's degree + transcripts
  • CV highlighting technical skills, projects
  • Statement of Purpose (500-1000 words)
  • 2 reference letters
  • IELTS 6.5+ / TOEFL 90+
  • 60-min online exam: Calculus, Linear Algebra, Probability, Trigonometry, Python, Data Structures, Algorithms
  • Technical interview (if shortlisted)
Timeline
Strategy

Apply Immediately

Rolling = earlier decision. Reapply if rejected.

Prepare for Exam

Linear Algebra, Probability, Calculus, Python, basic ML.

Best Backup

€3,900/mo highest stipend. Use as safety while waiting for Erasmus.
