# Master Timeline — Erasmus Mundus Application 2027

## Overview
The Erasmus Mundus application cycle runs **October 2026 – January 2027** for programs starting **September 2027**. This timeline covers preparation, application, and post-acceptance steps for ALL 5 programs.

## You Can Apply to Max 3 Programs
Choose your **top 3** from the 5 shortlisted programs. Recommended combination:

| Priority | Program | Reason |
|----------|---------|--------|
| 1st | **EMAI** (AI) | Best fit for ML career goals |
| 2nd | **InterMaths** (Mathematics) | Strongest academic match for Stats degree |
| 3rd | **DEAI** (Data Engineering & AI) | Great for data engineering + AI focus |

---

## PHASE 1: Foundation (Now – Sep 2026)

### Jun 2026 — Start Here
- [ ] Read all 5 program files in this guide
- [ ] Choose your **top 3 programs** (you can only apply to 3)
- [ ] Visit each program's official website and note exact requirements
- [ ] Create a **spreadsheet** tracking deadlines, requirements, and document checklists

### Jul 2026 — Language Test
- [ ] Register for **IELTS Academic** (target: 6.5 overall, no band < 6.0)
- [ ] Start preparation (Cambridge books, British Council resources)
- [ ] If your university education was in English, check if you qualify for a waiver

### Aug 2026 — Document Collection
- [ ] Request **certified academic transcripts** from Dhaka College
- [ ] Request **degree certificate** (or provisional certificate)
- [ ] Get **English translations** of all academic documents (certified translator)
- [ ] Scan all documents in **color PDF** format
- [ ] Prepare **Europass CV** format: https://europa.eu/europass/en

### Sep 2026 — Recommenders & Motivation
- [ ] Identify **2-3 recommenders**:
  - 1 Statistics professor from Dhaka College
  - 1 ML Engineer supervisor from your job
  - 1 from volunteer/leadership role (optional)
- [ ] Contact recommenders with your CV, program details, and deadlines
- [ ] Start drafting **motivation letter(s)** — one per program
- [ ] Check each program's **application portal** — register early if open

---

## PHASE 2: Application (Oct 2026 – Jan 2027)

### Oct–Nov 2026 — Open Portals
- [ ] All programs open applications around October
- [ ] Register on each program's portal
- [ ] Complete online application forms (note: EDISS must be done in one session)
- [ ] For AISS: prepare **motivation video** and **portfolio**
- [ ] For EDISS: prepare **video answer**
- [ ] Finalize **motivation letters** — each tailored to the specific program
  - EMAI: emphasize AI/ML projects + statistics foundation
  - InterMaths: emphasize statistical modelling + mathematical interests
  - DEAI: emphasize data engineering + ML deployment experience

### Dec 2026 – Jan 2027 — Submit
- [ ] Take **IELTS test** (if not done already)
- [ ] Upload all documents to each portal
- [ ] Remind recommenders to submit letters before deadline
- [ ] **Submit 48+ hours before deadline** (servers crash on the last day)
- [ ] Track submission confirmations and application IDs

## PHASE 3: Results (Feb – Aug 2027)

### Feb–Mar 2027 — Interviews
- [ ] AISS may invite for **15-min interview** — prepare your AgriTech story
- [ ] EMAI may conduct interviews for shortlisted candidates
- [ ] Check email regularly (including spam folder)
- [ ] Respond to any requests within 24 hours

### Apr–May 2027 — Decisions
- [ ] Main list offers arrive (usually April–May)
- [ ] If offered: **accept within 3–5 days** (strict deadline)
- [ ] If on reserve list: wait for possible upgrade (could be as late as September)
- [ ] If rejected: assess options (self-funded path or next year)

### Jun–Jul 2027 — Acceptance & Visa
- [ ] Accept the offer formally
- [ ] Pay any required deposits (if applicable)
- [ ] Apply for **student visa** for your first host country
  - Typical documents: admission letter, passport, financial proof, insurance
  - Bangladesh embassies may take 4–8 weeks
- [ ] Arrange **accommodation** (university housing or private)
- [ ] Book **flights**

### Aug 2027 — Pre-departure
- [ ] Attend program orientation (online)
- [ ] Pack essentials (travel light — you'll move countries in 4–6 months)
- [ ] Notify current employer
- [ ] Prepare finances for first month (scholarship first payment in October)

---

## PHASE 4: Go! (Sep 2027)
- [ ] Arrive in your first host country
- [ ] Register at the university
- [ ] Open a local bank account (for scholarship payments)
- [ ] Attend welcome week
- [ ] Classes begin!

## Backup Strategy
If you don't get a scholarship (or are on a reserve list):
1. Some programs offer **self-funded** spots with lower tuition (e.g., AISS: €2,000/semester)
2. Apply to **DAAD EPOS** (deadlines often later, ~Aug–Sep)
3. Consider **MBZUAI** (UAE) — fully funded, rolling admissions, ML-focused
4. Apply for **Chevening** (UK) when it opens in August 2027
5. Consider deferring to the next cycle with a stronger application
