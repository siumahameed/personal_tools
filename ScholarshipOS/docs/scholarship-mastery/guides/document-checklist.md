## Document Templates

### 1. Certified Academic Transcripts

**What to request from Dhaka College:**
- Official transcripts for all semesters (BSc Statistics)
- Must be on official letterhead, stamped and signed
- Must include: course names, credits/ECTS, grades, cumulative CGPA
- Request 5+ copies (you'll need original + 2 per application)

**Process:**
1. Visit Dhaka College registrar's office
2. Submit written application (template below)
3. Pay processing fee (~500-1,000 BDT)
4. Collect in 5-10 working days

> **Application template:**
> To,
> The Registrar
> Dhaka College, University of Dhaka
>
> Subject: Request for Certified Academic Transcripts
>
> Dear Sir/Madam,
>
> I, [Your Name], was a student of BSc (Honours) in Statistics at Dhaka College from [Year] to [Year], bearing Roll No: [Number], Registration No: [Number]. I require [5] copies of my official certified academic transcripts for higher study scholarship applications abroad.
>
> I have enclosed the required fee of [Amount] BDT. Kindly issue the transcripts at your earliest convenience.
>
> Sincerely,
> [Your Name]
> [Phone Number]

### 2. Degree Certificate

**If provisional certificate available:**
- Get certified copy + English translation
- Apply for full certificate if not yet received

**If degree not yet conferred:**
- Request "Letter of Expected Graduation" from college
- Use for conditional admission

### 3. English Translations

**Requirements:**
- Must be done by a certified/notarized translator
- Each document: original + translation stapled together
- Translator's seal/stamp on every page
- Common in Dhaka: NADRA, notary public offices

### 4. Europass CV

**Format:** https://europa.eu/europass/en
**Sections to fill:**
- Personal information + photo
- Education (BSc Statistics, Dhaka College)
- Work experience (Aspirant ML Engineer)
- Skills (Python, FastAPI, Streamlit, SQL, ML frameworks)
- Languages (Bengali, English)
- Projects (VocabPro, StatWise, AgriTech)
- Certifications (if any)

### 5. Passport

- Minimum 18 months validity from Sep 2027
- Color scan of bio-data page
- If renewing: start process now (4-6 weeks in Bangladesh)

## Recommender Strategy

| Recommender | Type | For | Key Points |
|-------------|------|-----|------------|
| 1. Statistics Professor (Dhaka College) | Academic | All programs | Can attest to quantitative skills, math foundation, academic performance |
| 2. ML Engineer Supervisor | Professional | All programs | Can attest to ML skills, production deployment, teamwork, growth potential |
| 3. Volunteer/Project Lead (optional) | Character | Fulbright, Chevening | Can attest to leadership, community impact, initiative |

**How to approach recommenders:**
1. Email them with: your CV, program details, deadlines, key points to highlight
2. Offer to draft bullet points about your work together
3. Give them 4+ weeks notice
4. Remind them 2 weeks before deadline
5. Send thank-you note after submission

## Document Scanning Guide

| Detail | Specification |
|--------|--------------|
| Scanner resolution | 300 DPI minimum (600 DPI preferred for transcripts) |
| Color | Color for all original documents |
| Format | PDF (single file per document) |
| File naming | `LastName_FirstName_DocumentType.pdf` |
| Max file size | Check each portal (typically 2-10 MB) |
| Merged files | Some portals accept single merged PDF |
