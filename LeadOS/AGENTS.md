# Project Rules for AI Assistant

## Server Policy (MANDATORY)
- NEVER start, restart, or launch any server without my explicit permission.
- Only start a server when I tell you to or ask you to test something that requires it.
- If you need to verify a server-related fix, ask me first before starting anything.
- When I say "stop the server" or "kill the server", immediately terminate any server processes.

## Communication
- Be concise. Answer in 1-4 lines unless I ask for detail.
- Don't add explanations after making changes unless I ask.
